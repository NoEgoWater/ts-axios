# ts-axios
使用typescript实现axios

## typescript library starter

### 项目使用typescript library starter创建
集成了很多优秀的开源工具
- 使用rollup.js帮助我们打包
- 使用Prettier和TSLint帮助我们格式化代码及保证代码风格一致
- 使用TypeDoc帮助我们自动生成文档并部署到GitHub pages
- 使用Jest帮助我们做单元测试
- 使用Commitizin帮助我们生成规范的提交注释
- 使用Semantic release帮助我们管理版本和发布
- 使用Husky帮助我们更简单的使用git hooks
- 使用Conventional Changelog帮助我们通过代码提交信息自动生成change log
